# Silkyware Landing Page

An interactive landing page featuring sleek scroll animations, a dynamic Apple-style dock with smooth magnification effects, and a premium aurora background light effect.

## Project Structure

This project has been split into separate files for clean organization and easy deployment:

- `index.html` - The markup and structure of the landing page.
- `style.css` - Custom CSS styling for the sections, aurora background, responsive dock, and animations.
- `script.js` - JavaScript handling the scroll animations, perspective transformations, interactive dock, and world map SVG drawing.
- `world-map.svg` - A customized vector dotted world map used as the background layout for connectivity paths.

## Features

- **Aurora Background Light Effect**: Smooth moving gradients that create a premium feel.
- **Scroll Animations**: Transforming 3D perspective scroll element that scales down smoothly.
- **World Map Connectivity**: An animated vector world map that dynamically projects latitude and longitude coordinates into curves and pulsing node paths when scrolled into view.
- **Testimonials Marquee**: An infinite, auto-scrolling horizontal marquee track featuring styled developer testimonial cards with a pause-on-hover effect and gradient fade masks.
- **Interactive Apple-Style Dock**: An interactive navigation dock at the bottom of the screen with fluid item scaling on hover and neighbor-grow effects.
- **Responsive Layout**: Works beautifully on mobile, tablet, and desktop screens.


## Quick Start

Simply open `index.html` in any web browser to view the page.

To deploy this project to GitHub Pages:
1. Initialize a Git repository.
2. Commit these files (`index.html`, `style.css`, `script.js`).
3. Push to a GitHub repository.
4. Enable GitHub Pages in your repository settings targeting the `main` branch.
