# Silkyware Hub

Silkyware is a sleek, modern desktop application designed for discovering, downloading, and managing software, PC games, and Android APKs with fast download speeds, active transfer controls, and effortless file organization.

## Features

### Dashboard
- **What it is**: A visually stunning homepage featuring dynamic aurora light effects and top featured releases.
- **How it works**: Displays highlighted releases across software, games, and APK categories upon launch. Gives you immediate access to popular and trending downloads with smooth visual feedback and glowing card highlights.

### Categorized App Libraries
- **What it is**: Organized browsing hubs dedicated to desktop software, PC games, and Android APK packages.
- **How it works**: Easily switch between dedicated category tabs to explore curated cards showing application icons, working status badges, release dates, and category tags.

### Real-Time Search & Instant Filtering
- **What it is**: An instant search bar built into every catalog view.
- **How it works**: Type any keyword into the search bar to immediately filter through titles and descriptions in real time, helping you find your desired software or game instantly without reloading.

### Interactive App Specs & Screenshot Carousel
- **What it is**: A detailed information modal for every item in the catalog.
- **How it works**: Click on any application card to open a full showcase window displaying detailed descriptions, version metadata, developer notes, system specifications, and an interactive screenshot carousel to preview the app before downloading.

### Smart Download Manager & Speed Monitor
- **What it is**: A live transfer control center for tracking and managing ongoing downloads.
- **How it works**: Keeps you informed with real-time download speeds (KB/s, MB/s), progress percentages, and estimated time remaining. Gives you full control to pause, resume, or cancel active downloads whenever needed. An active download badge on the sidebar keeps track of total background downloads.

### Installed App Manager & Quick Access
- **What it is**: A centralized library for managing all your downloaded files and software.
- **How it works**: Automatically logs completed downloads with installation timestamps and file names. Click "Open Folder" to jump straight to your downloaded files in your system file manager (Finder or File Explorer), or delete downloaded files directly from disk when you need to free up storage space.

### Customizable Download Settings
- **What it is**: Flexible storage management and application configuration options.
- **How it works**: Choose your preferred destination folder on your computer where downloaded apps and games will be saved into organized subdirectories. Silkyware remembers your settings across launches.

---

## How to Use

1. **Launch Silkyware**: Open the application to access the Dashboard and explore featured applications.
2. **Browse & Search**: Use the navigation sidebar or search bars to find software, games, or APKs.
3. **Preview Applications**: Click any card to preview screenshots, read specifications, and check working status.
4. **Download**: Click **Download Now** to start downloading. Track speed and progress in the **Downloads** tab.
5. **Access Downloaded Files**: Go to the **Installed** tab to open the download destination folder in your file explorer or manage saved files.
